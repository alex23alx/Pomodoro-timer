## Pomodoro Timer App

The project required me to create a HTML boilerplate to create the layout of the page. I created 3 buttons: start, pause and stop as instructed. When the timer hit 1 minute the clock turns red. I then used CSS to style the page. I followed instructions and made the buttons red, green and yellow and made the clock turn yellow when the mouse is hovered over it. Lastly I used JavaScript to implement functionality. When the clock hits 0 the user is alerted to take a break.
